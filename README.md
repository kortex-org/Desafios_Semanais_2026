# Desafios Semanais 2026

## Como enviar sua resolução

Se você quiser compartilhar a sua solução com a gente, é só seguir o padrão de pastas do repositório:  
Para cada semana, existe uma pasta de `resoluções/`. Você deve criar uma subpasta com o seu nome (ou nick) e dentro dela colocar seus arquivos.

**Importante**: Você pode fazer as questões na linguagem que preferir! Inclusive, se quiser praticar mais de uma, sinta-se à vontade para enviar a resolução da mesma questão em diferentes linguagens.

**Siga este passo a passo:**
1. Vá até a pasta da semana que você resolveu (ex: `Semana 1`).
2. Abra a pasta `resoluções/`.
3. Crie uma pasta com o seu identificador (ex: `resoluções/Maria_clara/`).
4. Adicione os seus arquivos de código dentro da sua pasta (ex: `q1.py`, `q2.c`, etc.).

**A estrutura deve ficar assim:**

```text
📁 Semana 1
└── 📁 resoluções
    ├── 📁 vanel
    │   ├── 📄 q1.py
    │   └── 📄 q2.c
    │   └── 📄 q3.cpp
    │
    └── 📁 seu-nome  <-- Crie a sua pasta aqui
        ├── 📄 q1.py   <--- Mesma questão
        └── 📄 q1.c   <--- Em linguagens diferentes
        └── 📄 q2.c
```
